Tutorial

El siguiente tutorial demostrara como definir una clase y luego instanciarla en un objeto.

Que es una clase?

Una clase es una plantilla de un objeto definiendo los atributo(s) y el comportamiento(s) de esta. Cuando se habla de atributos se hace referencia a propiedades, cuando se habla de comportamiento se hace referencia a metodos que pueden ser ejecutados.

Tanto las propiedades como los metodos de una clase pueden ser de 4 tipos:

  • publicos
  • privados
  • protegidos
  • estaticos

Que es una objeto?

Un objeto es la instancia de una clase, una clase puede multiples instancias y cada una de estas instancias tienen diferentes direcciones de memoria para sus propiedades por lo cual no existre relacion entre instancias, es decir las instancias de una clase no comparten informacion de ninguna forma.

Definicion de la clase

	
public class Persona{
	String nombre
	int edad
	int diasVida
	String estado;

	public Persona(edad){
		this.edad = edad;
		this.diasVida = this.edad * 365;
	}

	private void dormir(){
		this.estado = 'No molestar';
	}

	private void despertar(){
		this.estado = 'Necesita cafe';
	}

	private void  trabajar(){
		this.estado = 'Necesita dormir';
	}

	public void  vivirDia(){
		this.despertar();
		this.trabajar();
		this.dormir();
		this.diasVida++;
	}

	public void obtenerEstado(){
		return this.estado;
	}
}

Instanciar la clase

		
Persona erik = new Persona(33);

erik.vivirDia();
System.out.println(erik.obtenerEstado());
// salida: 'No molestar'
